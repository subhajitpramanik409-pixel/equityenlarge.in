# Equity Enlarge

Equity Enlarge is a stock market analysis and education platform for Indian traders.

We provide:
- Chart-based technical analysis
- Stock scanners and indicators
- Trading courses
- Premium newsletters
- Market research tools

This website is built using:
- GitHub Pages (hosting)
- Cloudflare (domain & security)
- HTML / CSS frontend
